BASICS (VARIABLE/PRINT)

  • Definition of Terms

    1. Variable - That thing that stores a value
    2. String - Value that has quotation marks.
    3. Integer - A whole number value
    4. Float - A numbered value with decimal
    5. Boolean - A value of either True or False
    6. List - Group of values
    7. Elements - Values inside a list
    8. Index - The position of a value in a list

  • Variable

    1. variable = "123"

      Holds a string 123

    2. variable = 123

      Holds an integer 123

  • Print

    1. print("123")

      Output:
      123

    2. variable = "abc"
      print(variable)

      Output:
      abc

    3. variable = "abc"
      print("variable")

      Output:
      variable

  • Printing Blank Line

    \n can be used to print blank line. Can be stored in variable "\n" or can be printed directly print("\n")

  • Updating Variable

    1. variable = "abc"
      variable = "def"
      print(variable)

      Output:
      def

    2. variable = "abc"
      print(variable)
      variable = "def"
      print(variable)

      Output:
      abc
      def

    3. variable_1 = "abc"
      variable_2 = variable_1

      variable_2 holds a string abc

  • Shortcut to updating and creating string Variables

    1. variable_1 = "Today is a great day"
      variable_1 = variable_1.replace("Today", "Yesterday")
      print(variable_1)

      Output:
      Yesterday is a great day

    2. variable_1 = "Today is a great day"
      variable_2 = variable_1.replace("Today", "Yesterday")
      print(variable_2)
      print(variable_1)

      Output:
      Yesterday is a great day
      Today is a great day

    3. variable = "Today is a great day"
      print(variable.replace("Today", "Yesterday"))

      Output:
      Yesterday is a great day

  • Using + on Variable and Print

    1. variable = "abc" + "def"

      Holds a string abcdef

    2. variable_1 = "abc"
      variable_2 = "def"
      print(variable_1 + variable_2)

      Output:
      abcdef

  • Numbers

    1. variable = 1
      print(variable + 1)

      Output:
      2

    2. variable_1 = 1
      variable_2 = 3
      variable_3 = variable_1 + variable_2

      variable_3 holds an integer 4

      Same principle for +-*/

  • F-strings

    1. print(2 + "eggs")
      ERROR

    2. print(f"{2} eggs")

      Output:
      2 eggs

    3. variable = 2
      print(f"{variable} eggs")

      Output:
      2 eggs

    4. variable = f"{2} eggs"
      print(variable)

      Output:
      2 eggs

    5. print(f"{1+1} eggs")

      Output:
      2 eggs

    6. print(f"{2} eggs for {10} pesos!")

      Output:
      2 eggs for 10 pesos!

  • Boolean and Negator

    1. variable = True

      variable holds a boolean True

    2. variable = not True

      Variable holds a boolean False

    3. variable = True
      print(not variable)

      Output:
      False
